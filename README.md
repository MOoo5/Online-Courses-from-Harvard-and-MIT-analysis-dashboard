# Harvard & MIT Online Courses Analysis Dashboard

## Project Overview
This project focuses on analyzing online courses data from HarvardX and MITx platforms using Power BI to uncover insights about course performance, learner engagement, and certification trends.

The dashboard was created to help understand:
- Course distribution between HarvardX and MITx
- Enrollment and certification trends across subjects
- Gender and education level breakdown of learners
- Most popular course subjects and categories
- Participation and engagement rates across courses

---

## Problem Statement
The dataset contained hundreds of online courses across different subjects, institutions, and years.

The raw dataset was difficult to analyze directly because it contained:
- Dates stored as unformatted text strings
- Missing values in key columns
- Year column stored as numeric index instead of actual year
- Large number of percentage columns requiring normalization
- Long and inconsistent subject category names

The goal of this project was to clean and transform the dataset into meaningful visual insights that support data-driven analysis.

---

## Objectives
- Clean and preprocess raw online courses data
- Analyze enrollment, certification, and engagement trends
- Build an interactive dashboard using Power BI
- Create dynamic visualizations for easier exploration
- Extract business insights from the dataset

---

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Power BI Charts
- Slicers and Filters

---

## Dashboard Features
- HarvardX vs MITx course comparison
- Subject-wise course and enrollment analysis
- Gender and education level breakdown of participants
- Course launch trends over the years
- Certification rate analysis across courses
- Interactive filters and slicers

---

## Data Cleaning Process
The following preprocessing steps were applied:
- Converted Launch Date into proper date format
- Handled missing values in important columns
- Renamed year index values into actual years
- Simplified long subject category names
- Validated percentage and certification columns
- Improved formatting consistency across the dataset

---

## Key Insights
- MITx offers more courses compared to HarvardX
- Humanities and STEM subjects dominate the course catalog
- Certification rates remain relatively low across online courses
- Male learners represent the majority of participants
- Course launches and enrollments increased significantly over time
